# Info

This is the blog where I talk about basically everything I like, I'm a tinkerer and have a lot of hobby's, so a lot to talk about.

The blog was initially written in markdown and exported to html with [pandoc](https://pandoc.org/), I edit the html file with some new meta tags and I would fix up some things if necessary.

After that, I used straight markdown to make my posts and use a jekyll theme named [contrast](https://github.com/niklasbuschmann/contrast) to make it look beautiful.

Now I use the [Hugo](https://gohugo.io) framework to render the markdown posts in a theme I've made. There isn't a seperate repo for the theme, but you can just download it from this repo.
