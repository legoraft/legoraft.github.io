# Posts

This is a small experiment of mine. I like doing write-ups on cool things I've done or am doing. I've been toying with the idea to create a blog again, but I don't really have the time or energy to do that right now. I do want to be able to write a bit right now though. That's exactly what this repo is, just `posts`. It could evolve into a blog later on, but for now it just contains this readme and the `posts` directory.

Posts are formatted with toml frontmatter, because I've been having fun using [zola](https://www.getzola.org/) lately. Don't know if I'll keep it or change it out, but that's how it's done now.
