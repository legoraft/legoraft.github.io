Linux and time

November 29, 2022

Linux is an awesome project, it’s open source and you can customize everything, but is it really worth it? I’ve used Linux for about a year now and a lot of my time has gone towards my operating system instead of work or projects.

I started my Linux journey around this time a year ago, I switched to Manjaro and it worked amazingly, given that it was a bit more work than your regular Windows 10 install, which just works out of the box (even though you need to wait for an eternity to let it install). After about 4 months, I switched to Arch Linux and oh boy, that was quite the rollercoaster.

Arch Linux

After installing Arch, one of my first problems was sound. It took about half an hour to fix that. After that, everything was smooth sailing, I already had made an AwesomeWM config, so I got it to work somewhat out of the box. All my applications also weren’t a problem, as Manjaro was already Arch based, so I had everything worked out. After a day or so of configuring, my computer was ready to use. I started playing Minecraft again on my survival server with a friend and we had a blast, designing new stuff, hyping up every update and loving the Christmas time and tiny lockdown that we had over Christmas, after that schools started and motivation dropped slightly, but we still kept on playing even when time was scarce. Then my config started to get annoying.

Customization

I didn’t really like some stuff about my config and wanted to design more things on my own, from icon sets to file managers, so I started learning about design, which brought me into a huge rabbit hole of designing icons in Inkscape and trying to build a file manager in Python. I stepped out of the idea to design everything myself a few weeks back when I realized I needed a lot of time which I didn’t have and I accepted that not everything would be tailored towards me. Now, about a month later, I’m having some stability issues on Arch, Virtualbox is crashing my window manager sometimes and not everything is running as smoothly.

I’m looking at switching to Void Linux right now, which should be more stable and not require as much tweaking as Arch. The problem is that I’m taking a lot of time again to find new Linux stuff to work with, even though I’m not customizing something or trying to write things from scratch. The problem I’m running into is that Linux just asks to be tweaked with, which is amazing, but if you try to just play a game or get some work done, this can be a crux to work with. I’m still working on and off on my revamped Awesome config, so the tinkerer inside of me is never sleeping, but I’m trying to take it slow.

Conclusion

All together, it is fun to be able to customize your system and have all those options available, but Linux does take up a lot of time when you love to tinker with stuff. I’m currently working to finish a lot of school projects and research which is due around Christmas and I’m picking up Minecraft again, so stay tuned on a post about Minecraft. The real truth is that Linux is an amazing project, but that’s the twist. Other than Windows or MacOS, Linux is a project, not an out of the box operating system, and you need to learn to keep using your computer to have fun and be productive, not just a huge DIY project.
